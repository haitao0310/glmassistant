# 回归测试清单

> 每次涉及相关模块重构/改动后,按此清单验证。不是规则,是项目测试清单。

## 基础
- [ ] 应用能正常启动(无崩溃)
- [ ] 程序正常退出(无崩溃/无内存报错)

## 对话(P1 SSE)
- [ ] 发送一条消息,收到 GLM 回复
- [ ] 流式输出(逐字出现,非整段蹦出)
- [ ] 生成中点"停止"能中断,按钮回"发送"
- [ ] 中断后能再发(不卡"not idle")

## 多会话(P3 SQLite)
- [ ] 新建/切换/删除会话正常
- [ ] 重启程序,历史会话/消息还在

## Provider(P5)
- [ ] Provider 切换正常(若有多 Provider UI)
- [ ] 缓存命中/未命中日志正确(若启用 CachingProvider)

## 调试(P4)
- [ ] 调试 tab 显示历史请求
- [ ] 选中看 rawRequest/rawResponse JSON
- [ ] 重放能发出新请求

## 错误处理
- [ ] 改错 key → 友好提示"API key 错误"(非 raw 英文)
- [ ] 断网 → 网络错误提示 + 重试(看日志)
